# TIL

##Desturcuring(구조 분해 할당)
객체와 배열에 들어있는 값을 가져올때 사용

### [객체]

```
const jikwan = {
name : '지관',
age : 17
};
const people = ['철수', '재석']
```

### [기존]

```
var name = jikwan.name;
var age = jikwan.age;
var chulsu = jikwan[0];
var jaesuck = jikwan[1];
```

### [Destructuring 방식]

var {name, age} = jikwan;
console.log(name);

const [chulsu, jaesuck] = people;
