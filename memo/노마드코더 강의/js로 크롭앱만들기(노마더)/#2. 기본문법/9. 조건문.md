# 조건문

### 뭔가를 확인 해야할 때 주로 사용

## if & else

```
if(조건) {
    //조건이 참일 때 실행
}
else {
    // 조건이 거짓일 때 실행
}
```

## else if

-   ### else와 if를 합친 개념이라 보면 됨
-   ### 기존 else에서는 조건을 줄 수 없지만 else if는 줄 수 있음
-   ### if의 조건에 맞지 않을 때 else if의 조건이 참이면 실행
-   ### else는 else if와 if 모든 조건이 거짓일 때 실행
-   ### else if는 위치가 중요
